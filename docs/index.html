postkit

Version 0.5 shared Rust library for DCP Wizard, IMF Wizard, and DCP Doctor.

About

postkit provides the core building blocks for digital cinema post-production tools. MXF wrapping uses asdcplib-rs FFI bindings.

Modules

  • encode: JPEG 2000 encoding via Grok (subprocess or optional in-process FFI)
  • mxf_wrap: MXF track wrapping
  • hash: SHA-1/SHA-256 hashing
  • colour: Colour space conversion
  • loudness: EBU R128 measurement
  • job_queue: Background jobs
  • preferences: JSON preferences
  • profiles: Delivery presets
  • burnin: Subtitle burn-in
  • report: QC reports
  • watch: Watch folder automation
  • preview: Colour-managed DCP preview (not real-time)
  • ingest: Camera format detection (raw rejected)
  • conform: Timeline import
  • certificate: X.509 management
  • dolby_vision: HDR metadata
  • watermark: Visible text watermark (not forensic)
  • dcdm: DCDM creation
  • accessibility: Compliance heuristic scan
  • trailer: Trailer packaging

Usage

Add to your Cargo.toml:

[dependencies]
postkit = { git = "https://github.com/PostPerfection/postkit.git", tag = "v0.5.0" }
